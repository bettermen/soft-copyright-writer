# 软著申请表填写指南

## 基本信息区

### 软件全称
- **格式**：品牌名 + 功能描述 + 软件/系统/平台等后缀
- **规范**：✅「鱼探AI智能钓鱼助手软件」 ✅「松鼠账本个人智能记账系统」
- **禁止**：❌ 含字母版本号（如 V1） ❌ 通用词汇（如「管理系统」） ❌ 敏感词（如「监控」「破解」） ❌ 夸大词（如「最佳」「最强」）
- **查重**：提交前需在官网查重，避免与已登记软件高度重名

### 软件简称（选填）
- **格式**：不超过 4 个汉字
- **示例**：✅「鱼探AI」 ✅「松鼠账本」
- **禁止**：❌ 含英文字母 ❌ 超过 4 个汉字

### 版本号
- **格式**：「V」+ 数字（V1.0、V2.1、V3.0.1）
- **禁止**：❌ 含字母（如 beta、alpha） ❌ 含中文（如「测试版」） ❌ 直接跳大版本号（如首次申请直接 V5.0，会引发审查质疑）
- **建议**：首次申请建议 V1.0

### 分类号
- 参考《软件分类号表》，选择最匹配的分类
- 如不确定，选「30900-0000」（其他应用软件）

## 开发信息区

### 开发完成日期
- **规则**：
  - 不能晚于申请日期
  - 企业申请：不能早于企业成立日期
  - 必须早于或等于「首次发表日期」
- **格式**：YYYY-MM-DD（如 2026-01-15）

### 首次发表日期
- **规则**：
  - 不能早于开发完成日期
  - 如未发表，填写「未发表」
  - 发表即已向公众提供（上架应用商店、公开发布等）
- **注意**：发表日期和完成日期之间的间隔不宜过长（建议 6 个月以内）

### 开发方式
- **选项**：独立开发 / 合作开发 / 委托开发 / 职务开发
- **注意**：
  - 合作开发需提供合作协议
  - 委托开发需提供开发合同 + 权利归属证明
  - 职务开发需提供单位授权书（企业申请）

## 软件技术特点

### 软件用途和技术特点（200-500 字）
- **撰写要点**：
  - 客观描述软件功能和用途
  - 列出核心技术特点（3-5 条）
  - 避免营销化语言
  - 与设计说明书内容一致

**示例**：
```
本软件为一款面向个人用户的智能记账管理工具，主要功能包括：
1. 多维账目记录：支持收入、支出、转账等多种账目类型，用户可自定义分类标签；
2. 智能分类：基于规则引擎自动识别账目类别，减少手动操作；
3. 数据可视化：通过图表展示月度/年度收支趋势，辅助用户财务决策；
4. 预算管理：支持设置分类预算，超支自动提醒；
5. 数据导出：支持导出 CSV/PDF 格式报表。
```

### 硬件环境
- **示例**：CPU Intel Core i3 及以上、内存 4GB 及以上、500MB 可用存储空间
- **Web 应用**：写明浏览器版本要求

### 软件环境
- **示例**：Windows 10 64位及以上 / macOS 12 及以上 / iOS 15 及以上 / Android 10 及以上
- **Web 应用**：Chrome 90+ / Safari 14+ 等

### 编程语言
- 列出主要开发语言，如：Python 3.9 / TypeScript 4.5 / Java 11
- 与实际源代码一致

### 源代码行数
- 填写源代码总行数（不含空行、注释、第三方库）
- 与提交的源代码文档对应

## 申请人信息区

### 个人申请
- 姓名、身份证号、地址、联系电话
- 签名必须手写或使用合法电子签名

### 企业申请
- 企业全称（与营业执照一致）
- 统一社会信用代码
- 联系人姓名、电话、邮箱
- 加盖公司公章

### 合作开发
- 填写所有著作权人信息
- 署名顺序需与合作开发协议一致
- 所有著作权人签字/盖章

## 常见填写错误

| 错误 | 后果 | 修正 |
|------|------|------|
| 软件名称含 V1.0 | 形式审查不通过 | 将版本号从名称中移除，单独填写在版本号字段 |
| 开发日期晚于申请日期 | 逻辑错误，退回 | 修正为早于申请日的日期 |
| 企业成立日期晚于开发日期 | 逻辑错误，退回 | 确保开发日期在企业成立之后 |
| 软件简称超过 4 汉字 | 格式不符 | 缩减为 4 汉字以内 |
| 首次发表日期早于开发日期 | 逻辑错误 | 调整为开发日期之后或填写「未发表」 |
| 用途描述过于模糊 | 审查质疑 | 补充具体功能描述 |
| 编程语言与代码不符 | 一致性审查不通过 | 修改为实际代码使用的语言 |

## 在线申请操作流程

1. 访问中国版权保护中心官网：https://www.ccopyright.com.cn
2. 注册账号并完成实名认证
3. 进入「软件著作权登记」系统
4. 按指引在线填写申请表
5. 下载打印申请表，签字/盖章
6. 扫描为 PDF 上传
7. 上传所有附件材料（设计说明书、源代码文档、身份证明等）
8. 确认提交，等待受理通知（约 2 周）
