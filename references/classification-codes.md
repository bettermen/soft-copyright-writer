# 软件分类号参考表

## 说明

软件分类号用于在申请表中标识软件的类别。选择与你的软件功能最匹配的分类号。如不确定，选「30900-0000」（其他应用软件）。

## 常用分类号

| 分类号 | 类别名称 | 适用场景 |
|--------|---------|---------|
| 10100-0000 | 操作系统 | 桌面/服务器/嵌入式操作系统 |
| 20100-0000 | 数据库管理系统 | 关系型/非关系型数据库 |
| 30101-0000 | 办公软件 | OA、文档处理、协同办公 |
| 30102-0000 | 企业管理软件 | ERP、CRM、HRM、进销存 |
| 30103-0000 | 财务管理软件 | 记账、报税、财务分析 |
| 30104-0000 | 教育软件 | 在线教育、题库管理、学习平台 |
| 30105-0000 | 医疗健康软件 | 医疗管理、健康监测、电子病历 |
| 30106-0000 | 电子商务软件 | 网上商城、支付系统、供应链 |
| 30107-0000 | 社交软件 | 即时通讯、社区、直播 |
| 30108-0000 | 游戏软件 | 手游、端游、页游 |
| 30109-0000 | 工具软件 | 系统工具、开发工具、设计工具 |
| 30110-0000 | 多媒体软件 | 音视频处理、图像处理、动画 |
| 30111-0000 | 安全软件 | 杀毒、防火墙、加密、审计 |
| 30112-0000 | 物联网软件 | IoT 平台、智能家居、设备管理 |
| 30113-0000 | 人工智能软件 | 机器学习平台、NLP、CV、推荐系统 |
| 30114-0000 | 大数据软件 | 数据分析、数据可视化、BI |
| 30115-0000 | 区块链软件 | 区块链平台、数字钱包、智能合约 |
| 30200-0000 | 嵌入式软件 | 单片机、工控、车载系统 |
| 30300-0000 | 中间件 | 消息队列、API 网关、服务治理 |
| 30900-0000 | 其他应用软件 | 无法归入以上分类的软件 |

## 微信小程序分类

| 分类号 | 适用场景 |
|--------|---------|
| 30101-0000 | 办公类小程序（OA、协作工具） |
| 30103-0000 | 记账/财务管理类小程序 |
| 30104-0000 | 教育类小程序 |
| 30106-0000 | 电商类小程序 |
| 30109-0000 | 工具类小程序 |
| 30110-0000 | 多媒体/相机类小程序 |
| 30900-0000 | 其他未分类小程序 |

## 选择建议

1. 先确认软件的核心功能属于哪个大类
2. 如果跨多个类别，选最核心的功能对应的分类号
3. 如果以上分类都不匹配，选 30900-0000
4. 分类号不影响审查结果，仅用于统计归类
